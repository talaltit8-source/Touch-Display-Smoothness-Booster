# 📱 Touch & Display Smoothness Booster

A lightweight Magisk/KernelSU module built to maximize touchscreen polling rates, eliminate scrolling latency, and force full GPU hardware acceleration for UI rendering.

Developed by **@dr.tit** 🚀

---

## 🚀 Features
- **Ultra-Response Touch:** Increases screen polling rates for instant action in competitive games (PUBG, COD, etc.).
- **Friction Optimization:** Perfectly tunes UI fling and scroll friction for a buttery-smooth scrolling experience.
- **HW Acceleration:** Forces the system to render layouts using hardware acceleration to prevent micro-stutters.

---

## 🛠️ Installation
1. Download the `Touch-Smoothness by Dr.tit.zip` from this repository.
2. Flash it via **Magisk** or **KernelSU**.
3. Reboot your device to apply the improvements.
